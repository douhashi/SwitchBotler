# ロードマップ / マイルストーン

## マイルストーン（案）

| フェーズ | ゴール |
|---|---|
| **M0: 雛形** | `create-tauri-app` で React+TS 雛形生成、トレイ常駐の骨組み |
| **M1: 認証** | Token/Secret 登録 UI + セキュアストレージ保管 + 署名生成（Rust） |
| **M2: 読み取り** | デバイス一覧・ステータス取得と表示 |
| **M3: 操作** | コマンド送信（ON/OFF・カーテン等）+ シーン実行 |
| **M4: 磨き込み** | トレイメニュー・お気に入り・エラーハンドリング整備 |
| **M5: 配布** | GitHub Actions で各 OS インストーラを自動ビルド・署名・リリース |

## 未決事項 / 検討ポイント

- 対応デバイスの優先順位（どの機種から手厚くサポートするか）
- ステータスのポーリング間隔（レート制限 10,000/日との兼ね合い）
- Webhook 対応を入れるか（ローカルサーバ常駐の是非）
- モバイル版を最初から視野に入れて UI をレスポンシブ設計するか
