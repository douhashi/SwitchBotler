お気に入り＝ドロップ先（デバイス／シーン）

お気に入りはボタン（ピン）ではなく場所。カードをドラッグして移すと登録され、落とした位置がそのまま並び順になる（外へ出せば解除）。登録は「移動」なので、その他のセクションからは消える。この順番がトレイポップアップの表示順になる。

この後の進め方

この 4 画面で方向性が固まれば、React + Tailwind のコンポーネントとして実装に落とします（トークンは MASTER.md をそのまま Tailwind theme に展開）。各画面で調整したい点（余白・情報量・アイコン・並び順など）があれば、画面単位で直します。
