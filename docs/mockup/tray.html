実装メモ（提案）
・ フル操作: tray-popover.tsx の hasPowerToggle フィルタを外し、deviceInteraction() で toggle=Switch / detail=「>」/ press=「押す」に分岐（デバイスカードと同一ロジックを共通化）。
・ 「>」の遷移先: メインウィンドウを表示しつつ該当デバイスの詳細へ。現状 show_main_window(view) は view しか渡せないため、デバイス id を渡せるよう IPC を拡張（navigate イベントに id を載せる）。ここが唯一の非自明ポイント（実装: 中）。
・ 幅: トレイウィンドウ幅を 288→360px 程度に。tray-app.tsx の setSize の幅を更新（window.innerWidth 依存を固定幅へ）。
・ 可変高＋上限: ResizeObserver で内容高に追従する現行に Math.min(h, MAX) を追加。デバイス/シーンの各リストに max-height + overflow-y: auto（デバイス≒6行 / シーン≒3行）。
・ UX 注意: 小さなポップオーバー内の入れ子スクロールは扱いに注意（overscroll-behavior: contain で親への伝播を抑止）。片方だけにする「単一スクロール案」も可能（好みで選択）。
