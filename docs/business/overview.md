# プロジェクト概要

## SwitchBotler とは

**SwitchBotler**（スイッチボトラー）は、SwitchBot Cloud API v1.1 を利用して SwitchBot デバイスを Windows / macOS / Linux から操作するデスクトップアプリケーション。

名前は **SwitchBot + Butler（執事）** の造語で、「デバイスに仕えさせる執事」というコンセプトを表す。公式非公認の **OSS** として開発する。

## ポジショニング

- SwitchBot 公式アプリはスマホ前提。**PC の作業机から手を離さずにデバイスを操作したい**というニーズを埋める。
- システムトレイ常駐 + グローバルショートカットで、「ボタン一つ / キー一発」で操作できる軽量ユーティリティを目指す。
- 複雑な自動化（Home Assistant 等）ほど重厚ではなく、**手元の即時操作**にフォーカスする。

## ターゲットユーザー

| ユーザー像 | 使い方の例 |
|---|---|
| デスクワーク中心の SwitchBot ユーザー | PC 作業中にトレイから照明・エアコン・カーテンを操作 |
| 開発者 / パワーユーザー | グローバルショートカットやシーン実行を日常的に使う |
| マルチ OS 環境の人 | 会社 Windows・自宅 Mac・自宅サーバ Linux で同じ UI を使いたい |

## コア機能（MVP）

1. **認証設定** — SwitchBot アプリで発行した Token / Secret を登録・安全に保管
2. **デバイス一覧取得** — 登録済みデバイスと仮想赤外線デバイスを表示
3. **デバイス操作** — ON/OFF、カーテン開閉、赤外線コマンドなどの送信
4. **ステータス表示** — 温湿度計・ハブなどのセンサー値を表示
5. **シーン実行** — SwitchBot アプリで作成したシーンをワンクリック実行
6. **トレイ常駐** — システムトレイから主要操作にアクセス

## 将来的な拡張候補（Post-MVP）

- **グローバルショートカット** — アプリ非フォーカス時もキーで操作
- **お気に入り / クイックアクション** — よく使う操作をピン留め
- **Webhook 受信** — デバイス状態変化のリアルタイム反映（要ローカルサーバ）
- **モバイル版** — Tauri v2 のモバイル対応を活かし iOS / Android へ展開
- **複数アカウント対応**
